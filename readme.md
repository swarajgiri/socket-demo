### Socket server and client demo

### Requirements
- Nodejs v8.11.1
- yarn v1.6.0

### Installation
`yarn`

### Start app
- `yarn start`
- Open [http://localhost:8080/](http://localhost:8080/)
- Open [http://localhost:8083/api/people/AAAA](http://localhost:8083/api/people/AAAA) to get response from api
